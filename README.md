# Latent–Quantum Triage for Pneumonia Detection from Chest X-Rays

**Short description:** A reproducible hybrid quantum–classical framework for pneumonia detection, uncertainty estimation, calibration analysis, and selective triage using compact latent representations from chest X-ray images.

## Overview

This repository contains the Colab-ready implementation accompanying the manuscript:

**Latent Representation Learning and Quantum-Enhanced Uncertainty for Decision-Centric Pneumonia Triage from Chest X-Ray Images**.

The project moves beyond accuracy-only pneumonia classification and focuses on **decision-centric medical AI**. It combines compact latent representation learning, a hybrid latent–quantum transformation, Monte Carlo dropout–based uncertainty estimation, calibration analysis, and entropy-driven selective prediction for triage.

## Main contributions

- Compact latent representation learning for chest X-ray classification.
- Hybrid latent–quantum feature transformation for richer representation geometry.
- Uncertainty-aware inference using Monte Carlo dropout.
- Calibration analysis using reliability diagrams and expected calibration error (ECE).
- Selective prediction and triage using entropy-based referral of uncertain cases.
- Reproducible notebook pipeline with automated generation of figures, tables, and logs.

## Notebook workflow

The notebook is organized as a full experimental pipeline:

1. **Environment setup**  
   Installs dependencies, sets seeds, mounts Google Drive, and prepares output folders.

2. **Data loading**  
   Loads the PneumoniaMNIST dataset and prepares train/validation/test splits.

3. **Utility functions**  
   Defines logging, plotting, metrics, calibration, and artifact-saving utilities.

4. **Model definitions**  
   Implements the baseline CNN, latent-only model, quantum layer, and latent–quantum hybrid model.

5. **Training utilities**  
   Standardizes training, validation, checkpointing, and metric tracking.

6. **Model training**  
   Trains the baseline, latent-only, and latent–quantum models.

7. **Test evaluation**  
   Produces confusion matrices, reliability diagrams, and the main comparison table.

8. **Latent space analysis**  
   Uses PCA and t-SNE to visualize learned latent structure.

9. **Uncertainty and triage analysis**  
   Applies MC dropout and derives confident vs uncertain decisions.

10. **Temperature scaling**  
    Evaluates post-hoc calibration effects.

11. **Coverage–risk analysis**  
    Studies the trade-off between coverage and error under selective prediction.

12. **Latent dimensionality and quantum ablation**  
    Compares latent-only, random-projection, and latent–quantum variants across dimensions.

13. **Case-based triage analysis**  
    Extracts representative confident, uncertain, and error cases.

14. **Final summary**  
    Consolidates all results into publication-ready outputs.

## Method summary

Given a chest X-ray image, the framework:

- encodes the image into a low-dimensional latent representation,
- transforms part of that latent space using a parameterized quantum circuit,
- fuses classical and quantum-derived features,
- predicts pneumonia probability,
- estimates uncertainty using MC dropout,
- and defers uncertain cases through an entropy-based triage rule.

This makes the framework suitable for **reliable screening and decision support**, not only raw classification.

## Dataset

The experiments use **PneumoniaMNIST** from the **MedMNIST v2** collection.

Why this dataset is used:

- public and easy to reproduce,
- computationally efficient,
- appropriate for benchmarking lightweight medical imaging models,
- suitable for studying uncertainty, calibration, and triage behavior.
